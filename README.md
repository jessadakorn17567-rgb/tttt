# My Photobooth

โปรเจกต์นี้เป็นเว็บ Photobooth แบบง่าย ๆ สำหรับถ่ายรูปด้วยกล้องเว็บแคม แล้วตกแต่งด้วยสติกเกอร์และกรอบรูปก่อนบันทึกภาพ

## ฟีเจอร์หลัก

- เลือกกล้องที่ใช้งานได้จากรายการกล้อง
- ดูวิดีโอแบบเรียลไทม์จากกล้อง
- ถ่ายรูป แล้วใช้ Fabric.js แก้ไขภาพ
- เพิ่มสติกเกอร์อีโมจิและข้อความลงบนภาพ
- ใส่กรอบรูปโพลารอยด์หรือกรอบนีออน
- บันทึกภาพเป็นไฟล์ PNG
- ถ่ายใหม่ได้เมื่อไม่พอใจรูป

## วิธีใช้งาน

1. เปิดไฟล์ `index.html` ในเบราว์เซอร์ที่รองรับกล้อง (เช่น Chrome หรือ Edge)
2. ระบบจะตรวจสอบกล้องและแสดงรายการกล้องที่มี
3. เลือกกล้องจากเมนู แล้วกดปุ่มถ่ายรูป
4. หลังจากถ่ายแล้ว จะเข้าสู่โหมดแก้ไขภาพ
5. กดปุ่มกรอบรูปหรือสติกเกอร์ เพื่อเปิดเมนูตกแต่ง
6. เลือกกรอบหรือสติกเกอร์ที่ต้องการเพิ่ม
7. กด `บันทึกรูป` เพื่อเซฟไฟล์ PNG ลงเครื่อง
8. กด `ถ่ายใหม่` หากต้องการกลับไปถ่ายภาพใหม่

## โครงสร้างไฟล์

- `index.html` - หน้าเว็บหลักและโค้ด JavaScript ทั้งหมด
- `README.md` - คำอธิบายการใช้งานโปรเจกต์

## หมายเหตุ

- ต้องอนุญาตให้เว็บเข้าถึงกล้องในเบราว์เซอร์
- ถ้าไม่พบกล้องจะแสดงข้อความไม่พบกล้อง
- ใช้ Tailwind CSS สำหรับสไตล์ และ Fabric.js สำหรับจัดการภาพและวัตถุบนแคนวาส
